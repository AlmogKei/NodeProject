Toy API Documentation

# Method Endpoint Description
1 GET /toys Retrieve a list of all toys
2 GET /toys/search?s=lego Retrieve details of a specific toy by name
3 GET /toys/price?min=100
/toys/price?max=50 		Retrieve a list of min price 100
Retrieve a list of max price 50
4 GET /toys?limit=8 Retrieve list of 8 toys
5 POST /toys Create a new toy
6 DELETE /toys/:id Delete a toy by ID
